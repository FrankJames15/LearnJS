Storing Value

Now, let’s dive deeper and learn how to store values in JavaScript

Imagine yourself as a scientist in a chemistry lab.
So, if you look around the lab, you’ll find several cylinders Each cylinder contains different types of gases, starting from oxygen to nitrogen

But how do we differentiate which cylinder contains which gas?

To solve this problem, we get it labelled with the gas’ name on it.

Similarly, if we want to store some information, we store in our memory. And we’ve all the information labelled in order to avoid confusion. If someone asks our age, we don’t tell them our grandfather’s name. Do we? Definitely no!

In Programming, this storing container is called “Variable”.

So, variables are container which store values. We place values in these containers and then use those values by referring to the container’s name.
