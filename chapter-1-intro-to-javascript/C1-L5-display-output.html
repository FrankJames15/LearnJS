Displaying Output

Displaying output is very important especially when learning the language.
If we want to see if our code is correct, we can output data to check
There are 4 ways to display output:


1. Using Developer Console

We can access developer console simply by clicking Ctrl + Shift + J
By using the console.log() function we can write into the browserâ€™s developer console. 

console.log(5 + 2); // Output: 7

However, the output of console.log() cannot be seen on mobile devices because the developer console is only available on desktop browsers like Chrome or Firefox


2. Using Dialog Box

By using the alert() function, we can output data on a dialog box 

alert(1 + 1); // Output: 3

This should only be used for testing purposes as it will delete all existing HTML in the current page.


3. Writing to the Browser Window

By using the document.write(), we write to the content of the document 

document.write(5 + 5) // Output: 10
document.write() should only be used for testing purposes as it will delete all existing HTML in the current page.
