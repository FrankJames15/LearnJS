Our First Program

This lesson helps us to get started with JavaScript by showing you how to embed JavaScript code into a HTML page

Let us write our first program to display "Hello World"

<html>
    <body>
    <script> 
        document.write("Hello World!"); 
    </script> 
    </body> 
</html>

Step 2: Save the file as HelloWorld.html

Step 3: Open the file using any web browser

Output:

Hello World!

Example Explained

To insert a JavaScript into an HTML page, we use the <script> tag.

JavaScript code should be written inside the <script> tags ( <script> and </script>)

This helps your browser distinguish your JavaScript code from the rest of the code.

The document.write command is a standard JavaScript command for writing output to a page.

By entering the document.write command between the <script> and <script> tags, the browser will recognize it as a JavaScript command and execute the code line.

As the output show above, the browser writes Hello World! to the page
