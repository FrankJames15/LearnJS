How to Run JavaScript

Being a scripting language, JavaScript cannot run on its own. In fact, the browser is responsible for running JavaScript code.

When a user requests an HTML page with the JavaScript in it, the script is sent to the browser and it's up to the browser to execute it.

Tools we Need

One of major strengths of JavaScript is that it does not require expensive development tools

To start working with JavaScript all we need is a text/code editor to write our code and a browser to display the web pages we develop

Since JavaScript is an interpreted language inside the context of a web browser, we don't even need to buy a compiler

Editor

Various code editors are available which serve as speedy, powerful and flexible tools for JavaScript programming We can use any of the Editors as listed below:
  • Eclipse with JavaScript Development Tools
  • Microsoft Visual Studio
  • NetBeans
  • Notepad++
