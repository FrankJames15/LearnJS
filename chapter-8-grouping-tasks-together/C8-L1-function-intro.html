Introduction to Function

Let's say we want to create a program which will print the result of addition of 2 numbers.

We would create 2 variables to hold the values of 2 numbers and another variabe which will hold the value of the addition of the numbers. 

var x = 2;
var y = 3;
var z = x + y;
Followed by printing the 3rd variable as follows: 
console.log(z);

What if we've to calculate the results of several combinations like, 1 and 2, 4 and 5, 6 and 7 ?

Then creating a new variable every time and adding it in the third variable would be a lot of work to do.. Right?

What if I tell you that there's a way to just write the code once and use it for any combination of number?

That is, to write reusable code whih could be used anywhere in the program

The term we're looking for is called Functions.

Let's learn about it.
