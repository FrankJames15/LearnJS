What is JavaScript?

  • Lightweight and dynamic scripting language which is based on object oriented concepts

  • Created by Bredan Eich

  • JavaScript was previously used mainly for making webpages interactive such as form validation, animation, etc.

Nowadays, JavaScript is also used in many other areas such as server-side development, mobile app development and games, and much more!

Features:

  • Providess rich user interface
  • Increases the speed by validating user inputs before sending to the server and also reducing the load of the server
  • Supported by commonly used internet browsers, hence eliminating the need to install or download a new software
